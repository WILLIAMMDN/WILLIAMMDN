# Cyber Lab

This page documents the security practice area behind the profile.

## Operating Areas

- Linux command line operations.
- Kali Linux lab practice.
- Web security testing with controlled targets.
- Burp Suite workflows.
- Bash scripting and automation.
- Secure development habits.

## Lab Format

Each future writeup should follow this structure:

```text
Objective:
Environment:
Tools:
Method:
Finding:
Mitigation:
Reflection:
```

## Rules

- Work only in authorized environments.
- Document risk and mitigation, not just exploitation.
- Keep personal or sensitive data out of public writeups.
- Prefer defensive learning and system hardening.
